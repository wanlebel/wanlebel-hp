## 5. JavaScript実装

### サイドドロワーメニュー（各ページ共通）

各ページの `<script>` 内にIIFEでインライン実装。左からスライドするドロワー方式。

```javascript
(function() {
  var hamburger = document.getElementById('hamburger');
  var drawer = document.getElementById('drawer');
  var overlay = document.getElementById('drawer-overlay');

  function toggleDrawer(open) {
    drawer.classList.toggle('is-open', open);
    overlay.classList.toggle('is-visible', open);
    hamburger.setAttribute('aria-expanded', String(open));
    document.body.style.overflow = open ? 'hidden' : '';
  }

  hamburger.addEventListener('click', function() { toggleDrawer(true); });
  overlay.addEventListener('click', function() { toggleDrawer(false); });

  // Escapeキーで閉じる
  document.addEventListener('keydown', function(e) {
    if (e.key === 'Escape' && hamburger.getAttribute('aria-expanded') === 'true') {
      toggleDrawer(false);
      hamburger.focus();
    }
  });
})();
```

### スクロールフェードイン（各ページ共通）

```javascript
(function() {
  var io = new IntersectionObserver(function(entries) {
    entries.forEach(function(e) {
      if (e.isIntersecting) {
        e.target.classList.add('visible');
        io.unobserve(e.target);
      }
    });
  }, { threshold: 0.1 });
  document.querySelectorAll('.fade-in').forEach(function(el) { io.observe(el); });
})();
```

### GTMイベント（`assets/js/gtm-events.js`）

`data-cta` 属性を持つ要素のクリック時に `dataLayer.push({ event: 'cta_click', ... })` を発火。GTMコンテナIDが設定されると自動的に有効化。

### CTA設定（`assets/js/config.js`）

STORES予約URL・LINE URLなどを定数管理。URL変更時はこのファイルのみ編集すればよい設計（※現状 config.js はロードされているが、各 HTML で直接 href を記述している箇所も多いため、URL変更時は各 HTML も確認すること）。
