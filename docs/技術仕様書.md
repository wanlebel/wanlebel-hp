## 5. JavaScript実装

### ハンバーガーメニュー（各ページ共通）

```javascript
// 各 index.html の <script> 内にインライン実装
const menuBtn = document.getElementById('menu-btn');
const mobileMenu = document.getElementById('mobile-menu');
const menuIcon = document.getElementById('menu-icon');
// クリックでメニュー開閉・アイコン切替（≡ / ✕）
```

### スクロールフェードイン（各ページ共通）

```javascript
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) entry.target.classList.add('visible');
  });
}, { threshold: 0.1 });
document.querySelectorAll('.fade-in').forEach(el => observer.observe(el));
```

### GTMイベント（`assets/js/gtm-events.js`）

`data-cta` 属性を持つ要素のクリック時に `dataLayer.push({ event: 'cta_click', ... })` を発火。GTMコンテナIDが設定されると自動的に有効化。

### CTA設定（`assets/js/config.js`）

STORES予約URL・LINE URLなどを定数管理。URL変更時はこのファイルのみ編集すればよい設計（※現状 config.js はロードされているが、各 HTML で直接 href を記述している箇所も多いため、URL変更時は各 HTML も確認すること）。
