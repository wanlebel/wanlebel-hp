# サイト全体リデザイン実装プラン

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** パターンBモックのデザイン（黄色ベース・ティールアクセント・M PLUS Rounded 1c）を全6ページに適用する

**Architecture:** 共有CSS（`assets/css/style.css`）にデザイントークン・ヘッダー・フッター・共通コンポーネントを集約し、各ページはこれをリンクしてページ固有のスタイルのみインラインで持つ。Tailwind CDNを全廃してカスタムCSSに完全移行。TOPページはモックのレイアウトをそのまま採用し、サブページはヘッダー・フッターを差し替えてセクションを新デザインでリスタイルする。

**Tech Stack:** HTML5 / Vanilla CSS（カスタムプロパティ）/ Vanilla JavaScript / M PLUS Rounded 1c（Google Fonts CDN）

**JSテンプレート（全ページ共通 + TOP固有）:**
```html
<script>
  // ドロワーメニュー
  const hamburger = document.getElementById('hamburger');
  const drawer = document.getElementById('drawer');
  const overlay = document.getElementById('drawerOverlay');
  function toggleDrawer(open) {
    hamburger.classList.toggle('open', open);
    drawer.classList.toggle('open', open);
    overlay.classList.toggle('open', open);
  }
  hamburger.addEventListener('click', () => toggleDrawer(!drawer.classList.contains('open')));
  overlay.addEventListener('click', () => toggleDrawer(false));
  drawer.querySelectorAll('a').forEach(el => el.addEventListener('click', () => toggleDrawer(false)));

  // スクロールフェードイン
  const io = new IntersectionObserver((entries) => {
    entries.forEach(e => { if (e.isIntersecting) e.target.classList.add('visible'); });
  }, { threshold: 0.1 });
  document.querySelectorAll('.fade-in').forEach(el => io.observe(el));
</script>
```
