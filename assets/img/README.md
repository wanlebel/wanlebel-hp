# 画像ファイル配置ガイド

このディレクトリには、以下の画像ファイルを配置してください。

## 必要な画像ファイル

### トップページ用
- `hero.jpg` - サロン外観画像（推奨サイズ: 800x600px）
- `concept.jpg` - サロン内観画像（推奨サイズ: 800x600px）
- `gallery-1.jpg` ~ `gallery-6.jpg` - ギャラリープレビュー用画像（推奨サイズ: 400x400px、正方形推奨）

### ギャラリーページ用
- `gallery-1.jpg` ~ `gallery-9.jpg` - トリミング仕上がり例画像（推奨サイズ: 600x600px、正方形推奨）

### OGP画像
- `ogp.jpg` - トップページ用OGP画像（推奨サイズ: 1200x630px）
- `ogp-menu.jpg` - メニューページ用OGP画像
- `ogp-gallery.jpg` - ギャラリーページ用OGP画像
- `ogp-access.jpg` - アクセスページ用OGP画像
- `ogp-faq.jpg` - FAQページ用OGP画像
- `ogp-policy.jpg` - ポリシーページ用OGP画像

## 画像の要件

- ファイル形式: JPGまたはPNG
- カラーパレット: デザインシステムに準拠（#F5E1E6, #FFFBFB, #6B5E62, #C2E0EB）
- トーン: やさしく、落ち着いた雰囲気
- 内容: 犬、サロン内観、トリミングの様子など

## 注意事項

- 画像は実際のサロンの写真を使用してください
- 個人情報が写り込まないよう注意してください
- 画像の著作権を確認してください
